Software

Except where specified (in parentheses and italics), I am the primary–and in most cases only–contributing developer to the following open source research software projects.

kevlar Library

Alignment-free discovery of novel germline variants on simplex pedigrees. Implemented as a library with command line and Python APIs, and distributed as a Python package.

khmer Library

In-memory k-mer counting, filtering, graph traversal, and more. Implemented as a C++ library with Python bindings, and distributed as a Python package. (I'm a core contributor.)

tag Library

Genome annotation processing that supports graph-based representation of genome features and embraces a streaming processing paradigm. Implemented as a library with a Python API (as well as a minimal command line interface), and distributed as a Python package.

AEGeAn Toolkit

A collection of tools for analyzing and evaluating genome annotations, most notably ParsEval and LocusPocus. Implemented as a library with a C API.

GenHub

A companion to the AEGeAn Toolkit, providing a more comprehensive pipeline around LocusPocus for data retrieval, pre-processing, post-processing, and computing statistics on iLoci. Implemented in Python.

GenomeTools Library

A comprehensive toolkit for processing, transforming, and analyzing genome annotation data. Implemented as a library with a C API. (I'm a contributor.)
